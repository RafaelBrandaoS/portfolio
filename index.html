HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para criar páginas da web. Ele permite adicionar estrutura e formatação ao conteúdo da página, como títulos, parágrafos, links e imagens. HTML é interpretado pelo navegador e é o que permite que as páginas web sejam exibidas corretamente.

CSS

CSS (Cascading Style Sheets) é uma linguagem de folhas de estilo utilizada para separar a apresentação visual de um documento HTML de sua estrutura. Ele permite aos desenvolvedores aplicar cores, fontes, layouts e outros estilos a um documento HTML, sem precisar modificar o código HTML.

JavaScript

JavaScript é uma linguagem de programação de script usada para adicionar interatividade e dinamicidade a páginas web. Ele é interpretado pelo navegador e permite aos desenvolvedores criar scripts que podem manipular o conteúdo de uma página, como alterar o conteúdo de um elemento, validar formulários, criar animações e interagir com o usuário.

Python

Python é uma linguagem de programação de alto nível e de propósito geral, conhecida por sua sintaxe simples e legível. Serve para desenvolver uma ampla variedade de aplicações, incluindo desenvolvimento web, automação, análise de dados, inteligência artificial e ciência de dados. Python é mais utilizada em áreas como desenvolvimento web, análise de dados, machine learning e automação de tarefas.

MySQL

MySQL é um sistema de gerenciamento de banco de dados relacional (SGBD) de código aberto. Serve para armazenar, organizar e gerenciar grandes quantidades de dados de maneira eficiente. MySQL é amplamente utilizado em aplicações web e empresariais, oferecendo suporte robusto para consultas SQL (Structured Query Language).

Git

Git é um sistema de controle de versão distribuído. Ele permite que os desenvolvedores possam manter um histórico de todas as alterações feitas em um arquivo ou conjunto de arquivos ao longo do tempo, e reverter para versões anteriores se necessário. Ele também permite a colaboração entre múltiplos desenvolvedores, permitindo que eles trabalhem em arquivos ou projetos conjuntamente sem criar conflitos.

GitHub

GitHub é uma plataforma baseada na web para hospedar e colaborar em projetos que utilizam o sistema de controle de versão Git. Ele permite que os desenvolvedores armazenem, compartilhem e colaborem em projetos de software de forma pública ou privada.
